com.chartboost.sdk
Interface ChartboostDelegate

All Known Implementing Classes:
ChartboostActivity, ChartboostDefaultDelegate, ChartboostNativeActivity
public interface ChartboostDelegate

Implement the following methods to more finely control Chartboost's behavior. If you only want to implement some of the methods (as they are all optional), consider extending from ChartboostDefaultDelegate instead.

shouldPauseClickForConfirmation

boolean shouldPauseClickForConfirmation(Chartboost.CBAgeGateConfirmation callback)
Called after the user clicks on the ad. Developers can use this callback to do age confirmation and provide the result back to the SDK in the callback function by passing a boolean value

Parameters:
callback - Callback function to pass a boolean value to the callback function based on the user selection. Execute this callback only if you enable the age gate support.
Returns:
Default: 'false' (Return 'false' to disable the age gate and 'true' to enable age gate support)
