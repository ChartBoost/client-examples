public abstract class ChartboostActivity
extends android.app.Activity
implements ChartboostDelegate

Use this class as a base class for any activities of yours in order to simplify Chartboost integration. This class handles all of the necessary lifecycle method calls into Chartboost (Chartboost.onCreate(Activity, String, String, ChartboostDelegate), Chartboost.onStart(Activity), Chartboost.onStop(Activity), Chartboost.onDestroy(Activity), and Chartboost.onBackPressed(). Chartboost.startSession() is called for you in the onStart() method.

It uses itself as a delegate, implementing the ChartboostDelegate interface, allowing you to only override the methods that you specifically need finer control over.
